Architecture

Given our small model, there's almost a one to one correspondence to the previous developed work.

  • lm_archive_course: Implements Archive Generator.
  • lm_new_course.py: Implements Course Generator.
  • lm_new_lecture.py: Implements Lecture Generator.
  • A Course is implemented via a folder and a .ini file for its index
  • A Lecture is implemented via a whole html file, within a course directory
  • The global index is implemented via the data directory for the lectman suite.
  • lm_list.py: Will implemented the listing index capability.
