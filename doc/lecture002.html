Model

Given current status of project, and the only projected extension requirement, we essentially haven't got many entities or systems.

  • Lecture: Basic unit of lectman, it represents the notes on a given lecture.
  • Course: A set of lectures, which aggregates semantically a given set.
  • New Course Generator: Generates a given course with no lectures, associated with a given name.
  • New Lecture Generator: Generates a given lecture associated to a given course, and associates the lecture to it.
  • Archive Generator: Generates a presentable index, based in the course and its lectures.
  • Global index manager: A centralized reference of course repositories.
